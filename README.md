# image-converting-server

Image converting server built with Go.

## Getting Started

### Prerequisites

- Go 1.21 or higher

### Installation

```bash
go mod download
```

### Running the Server

```bash
go run main.go
```

The server will start on `http://localhost:8080`

### Endpoints

- `GET /` - Main endpoint
- `GET /health` - Health check endpoint
